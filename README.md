# Password Strength Checker

## Internship Project
This project is developed as part of the CODTECH Cybersecurity & Ethical Hacking Internship.

## Project Description
The Password Strength Checker checks whether a password is Weak, Medium, or Strong using Python.

## Features
- Checks password length
- Detects uppercase letters
- Detects lowercase letters
- Detects numbers
- Detects special characters

## Technology Used
- Python
